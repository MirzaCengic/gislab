
What's Next:

  * Copy the entire directory containing your new plugin to the QGIS plugin
    directory (~/.qgis2/python/plugins)

  * Compile the resources file using pyrcc4

  * Test the plugin by enabling it in the QGIS plugin manager

  * Customize it by editing the implementation file:
         webgisplugin.py

  * Modify your user interface by opening webgisplugin.ui in Qt Designer

  * You can use the Makefile to compile your resource files when
    you make changes. This requires GNU make (gmake)
