Super easy deployment of integrated GIS office with unlimited number of workstations in a few moments.

GIS.lab provides possibility to create integrated, fully equipped, easy-to-use, pre-configured, centrally managed, portable and unbreakable GIS office platform from one host machine running central server and unlimited number of diskless client machines. All software works out-of-box without any need of configuration or other behind a scene knowledge, allowing users to keep high focus on their GIS tasks.

GIS.lab architecture and services

  • super easy, fully automatic deployment and maintenance of server and client machines
  • very low requirements for client machine hardware and software
  • no limit of number of client machines
  • central management of all client machines, user accounts and user data
  • full client machine performance usage with no thin client glitches
  • unbreakable client machines

Features

Project created in desktop

  • raster and vector data processing and analysis
  • free access to raster base maps
  • print output composer

Automatic project publication on web

  • automatic web GIS publishing
  • web GIS data capturing and collaboration tools
